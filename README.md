# FinanceControlBot
Bot for accounting of finances in Google tables

Описание: основная идея заключается в хранении данных пользователя не где-то на сервере, а на его Google диске в таблице. Сам же бот предоставляет удобный интерфейс в Telegram для взаимодействия с таблицей: добавление расходов/доходов, настройка категорий трат/доходов, настройка счетов.
